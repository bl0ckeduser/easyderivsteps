I. Tips on input syntax


Although this program shows results in standard mathematical notation, expressions must be input in plain-text.

New: Since version 1.1, implicit multiplication shorthand notations like "5x^3" instead of "5*x^3" and "(1+x)(1+x^2)" instead of "(1+x)*(1+x^2)" are supported.

Here are some comparative examples of this syntax:

Plain-text
 Mathematical notation
x^3
5x^3
-e^-x^2
(1+x)(1+x^2)^2
e^(cos(x)sin(x))
(3x^4 + 5e^x) / (cos(x^x) log(x^2)) * sqrt(1 / cos(x) + arctan(x^x))

It is possible to use the "Preview" button to see how the currently-entered problem translates to standard mathematical notation:



Expressions containing irrecoverable syntax errors will not yield any preview, nor will they successfully differentiate.

II. Supported functions


This program "knows" the following functions, and therefore can compute derivatives involving them:

Function
 Description
arccos() Inverse cosine
arccot() Inverse cotangent
arccsc() Inverse cosecant
arcsec() Inverse secant
arcsin() Inverse sine
arctan() Inverse tangent
cos() Cosine
cot() Cotangent
csc() Cosecant
e^() Exponential
ln() Natural logarithm
log() Natural logarithm
sec() Secant
sin() Sine
sqrt() Square root
tan() Tangent

Parentheses around function arguments are mandatory.

III. Tips on reading output


Once you have chosen an expression and variable and clicked on "Differentiate", the program will perform its calculation work and (hopefully) soon enough produce complete steps. The produced results will include:

  • Expressions written out after each step, in black.
  • Descriptive comments explaining most of the steps, written in blue.
  • Changes made after a given step, highlighted in red.
